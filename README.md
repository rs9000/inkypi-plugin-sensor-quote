# Sensor & Quote Plugin for InkyPi

This plugin for InkyPi displays sensor data and an inspirational Zen quote. It's designed to be flexible, allowing you to source sensor data from either a remote device using SSH or a local file.

## Features

*   **Displays sensor data:** The primary function of the plugin is to display sensor readings from a temperature and humidity sensor, such as a BME680.
*   **Zen Quote:** Along with the sensor data, a random Zen quote is displayed.
*   **Two Data Modes:**
    *   **Remote (SSH):**  Fetch sensor data from a remote device.
    *   **Local (File):** Read sensor data from a local file.
