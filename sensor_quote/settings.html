Sensor & Quote Display Plugin

Displays sensor data and a zen quote. The sensor data can be fetched from a remote device via SSH or from a local file.

Data Source
Full path to the sensor JSON file. This can be a local path or a path on a remote system.
Remote SSH Configuration (Optional)

Leave the SSH Host and User fields blank to read from a local file path.

Optional. Hostname or IP address of the remote device (e.g., your-device, 192.168.1.100).
Optional. Username for the SSH connection (e.g., user, pi).

For Remote SSH Access

Make sure SSH key authentication is configured. Run on this device:

ssh-copy-id user@host
